# Portfolio

I've created this portfolio project to showcase my latest work and skills. Please enjoy.

My goal here is to promote the use of the JAMStack, as I'm a big fan of it,
so this site has been generated using `React-Static`, the blog content is written in `Markdown`, and asset optimisation will be automatically performed by `Netlify`.

I totally recommend this stack, in my opinion is the best I've used so far. Also, I used Netlify and Github Continuous Deployment pipeline.
